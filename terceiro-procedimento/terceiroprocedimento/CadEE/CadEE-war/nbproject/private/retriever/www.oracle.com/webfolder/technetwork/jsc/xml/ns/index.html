Introduction

All new Java related schemas are created in subdirectories of this directory. The canonical name for this directory is http://xmlns.jcp.org/xml/ns/ althought it's currently hosted in a different location. Note also that many of the Java schemas from java.sun.com are hosted in the same location, although their canonical names remain http://java.sun.com/xml/ns/....


Java EE Schemas

Currently the only schemas in this namespace are the Java EE 7 schemas at http://xmlns.jcp.org/xml/ns/javaee/index.html.
